Startseite¶

Ein Kurs aus dem Bereich Überfachliche Kompetenzen der HTWK-Leipzig von Felix Kaule und Stephan Schoenfelder

Es werden ausschließlich open source bzw kostenfreie Lösungen verwendet und dabei eine bessere Alternative zu Microsoft Word bzw. Excel aufgezeigt.

  1. Texte erstellen mit LaTeX Alternative zu Microsoft Word

  2. Literaturverwaltung mit Zotero Alternative zur (wenn überhaupt verwendeten) Literaturverwaltung in Microsoft Word

  3. Grafiken mit Inkscape Alternative zu Adobe Illustrator / Corel Draw

  4. Datenvisualisierung mit Python Alternative zu Microsoft Excel

Im letzten Teil werden allgemeine Hinweise zur Gestaltung wissenschaftlicher Arbeiten gegeben.
